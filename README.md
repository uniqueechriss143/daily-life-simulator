# Daily Life Simulator

This is a beginner Python project that simulates a person's daily life decisions.

## Features
- Wake up or stay asleep
- Choose morning routine
- Go to work, grocery, or stay home
- Grocery system with total cost and change

## Skills Used
- Python basics
- if / elif / else
- user input
- variables
- basic math operations
